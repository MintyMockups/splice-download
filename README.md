# NIXO Splice Ripper

Full package for ripping raw audio from Splice samples.

## Files
- index.html → Web version (GitHub Pages)
- splice_auto.py → Powerful Python + Playwright version (best success rate)

## How to use HTML
1. Host index.html on GitHub Pages
2. Open and paste Splice link

## How to use Python (Recommended)
```bash
cd splice-ripper
python splice_auto.py "https://splice.com/sounds/..."
```

Works on Termux, VPS, etc.
